FtpConnection

Package: jasononeil.ftp

This class sets up a new connection to an FTP server via PHP’s FTP functions.  Currently there is no Neko port.

This class should allow you to create a connection and log in, verify it’s working, and then you can use <getFileAt()> and <getDirAt()> to grab the <FtpFile> and FtpDir objects and manipulate those.

To get the root directory use this.getDirAt(‘/’);

server

(String) The domain name of the server, not including “ftp://” (Read Only).

Can only be set through the new() constructor.

Example

"ftp.jasononeil.com.au"     // valid
"mywebhost.com"         // valid
"localhost"             // valid
"203.101.58.7"          // valid

"ftp://ftp.jasononeil.com.au"   // invalid, should not include protocol
"mywebhost.com/"            // invalid, should not include trailing slash
"localhost:21"          // invalid, should not include port
