FtpDir

Package: jasononeil.ftp

Using an FtpConnection, this class lets you view or modify directories on an FTP server.

Given the path, this class will check if the directory exists.  If it does, then the properties of the object will be populated to match the directory.  If it doesn’t, you can use the mkdir() method to create it.

path

public var path(default,
null):String

(String) The path on the FTP server to this directory (Read Only).

If the directory exists, then all other fields will be populated accordingly.  If it doesn’t, then you can use the mkdir() method to make it.

This should include the leading slash, but not the trailing slash.

Example

"/"                 // valid
"/public_html"          // valid
"/public_html/admin"        // valid
"/backups"              // valid
"public_html"           // invalid, no leading /
"/public_html/"         // invalid, should be no trailing /
"/backups/myfile.html"      // invalid, is not a directory

Can only be set through the new() constructor.
