replaceElementWith

public static function replaceElementWith(originalElement:Xml,
?string:String,
?xml:Xml,
?iterator:Iterator<Xml>)

Replaces an element with one or more new Elements.  This is especially useful for “unwrapping” elements, for example, turning:

<title><test>My Page</test></title>

Into

<title>My Page</title>

You could do this by calling

replaceElementWithElements(element, element.iterator());

Parameters

originalElement(Xml) The element to get rid of
string?  (String) Text to replace it with
xml?  (Xml) XML to replace it with
iterator?  (Iterator<Xml>) List of XML objects to replace it with.

Returns

(Bool) Was the operation successful?
